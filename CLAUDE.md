# Landing Page

## Descrição
Landing page pessoal desenvolvida com React e TypeScript.

## Stack
- React + TypeScript
- Tailwind CSS
- Vite

## Estrutura
src/
├── components/
│   ├── Header.tsx
│   ├── Hero.tsx
│   ├── Features.tsx
│   ├── About.tsx
│   ├── Contact.tsx
│   └── Footer.tsx
├── hooks/
├── utils/
└── styles/

## Convenções
- Componentes: PascalCase (Button.tsx)
- Hooks: camelCase com prefixo use (useAuth.ts)
- Commits: em inglês, conventional commits

## Comandos
- npm run dev — desenvolvimento local
- npm run build — build de produção
- npm run lint — verificar código

## Seções da Landing Page
1. Header — navegação e logo
2. Hero — chamada principal e CTA
3. Features — principais funcionalidades/serviços
4. About — sobre mim/empresa
5. Contact — formulário ou informações de contato
6. Footer — links e copyright
